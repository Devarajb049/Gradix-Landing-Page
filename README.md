## What is this?

A **premium AI SaaS landing page** built during my Frontend Development Internship at **Gradix Tech**. The goal was to ship something production-ready — responsive, animated, performant, and visually sharp — using only vanilla frontend tools.

No frameworks. No bloat. Just HTML, CSS, JavaScript, and Tailwind done right.

> 🔗 **[gradix-landing-page.vercel.app](https://gradix-landing-page.vercel.app/)**

---

## Features

| | Feature | Detail |
|---|---|---|
| 🎨 | **Modern UI** | SaaS-grade design with clean hierarchy and intentional whitespace |
| 📱 | **Fully Responsive** | Mobile-first layout, tested across all breakpoints |
| ⚡ | **Optimized Performance** | Lightweight assets, smooth scroll, minimal render-blocking |
| ✨ | **Scroll Animations** | AOS-powered entrance effects on every section |
| 🧭 | **Conversion-Focused** | CTA placement, hero copy, and nav built to convert |
| ♿ | **Accessible** | Semantic HTML, ARIA labels, keyboard-navigable |

## Getting Started

```bash
# 1. Clone
git clone https://github.com/Devarajb049/Gradix-Landing-Page.git
cd Gradix-Landing-Page

# 2. Install dependencies (if using npm scripts)
npm install

# 3. Start dev server
npm run dev

# 4. Build for production
npm run build
```

> **No build step?** Just open `index.html` directly in your browser — it works out of the box.
